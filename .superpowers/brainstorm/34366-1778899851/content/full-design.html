Junior — Full Design

Hands-free AI guide assistant · Next.js · Claude claude-sonnet-4-6 · Vercel

Core Features

📄 Guide Creation
Upload PDF · fetch from URL · photograph physical doc (camera OCR). Claude extracts structured steps + attaches images with AI-generated crop bounding boxes.
🎙 Voice Session
Web Speech API input · Claude answers with full guide + current step context · Speech Synthesis output. Says "next"/"done" to advance, or asks anything freely.
🖼 Smart Image Display
Images surface automatically on step advance or on voice request ("What do I expect to see?"). AI crop overlay highlights the relevant region. User can still pinch/zoom freely.
📷 Progress Check
User takes a photo mid-task. Claude compares it to the current step and AI crop — gives go/no-go feedback ("your dough looks right, keep going").
🔗 Shareable + Forkable
Every guide gets a public slug (/g/abc123). Anyone with the link can run a session. Fork = saves a new copy under a new ID for customisation.
🍳 Cooking-First, General-Ready
Demo showcases a baking recipe. Architecture supports any domain — manuals, assembly guides, KTANE rulebook.

Data Model

Guide {
  id: nanoid            // public URL slug
  title: string
  source: 'pdf' | 'url' | 'camera'
  steps: Step[]
  fork_of?: string        // parent guide id
  created_at: timestamp
}

Step {
  index: number
  title: string
  content: string
  image_url?: string      // Vercel Blob URL
  crop?: { x, y, w, h }   // AI bounding box (% of image)
}

AI Design — Session Turn

System prompt: You are Junior, a hands-free guide assistant.
Guide: [full guide text]. Current step: [N of M] — [step content].
Speak naturally and briefly. When the user advances, confirm the new step aloud.
If the user asks to see something visually, reply with JSON action alongside your spoken response:
{ "action": "show_image", "step": N }
On camera photo: assess whether it matches the current step. Give go/no-go feedback.

Response format:
{ "speech": "...", "action": "show_image" | "advance" | null, "step": N }

Stack & Deployment

Next.js 14 (App Router)
Claude claude-sonnet-4-6 (vision)
Vercel KV (guide storage)
Vercel Blob (image storage)
Web Speech API (voice in/out)
pdf.js (PDF rendering)
Vercel (deploy)
No auth required. Guides identified by ID. Forks stored in localStorage until explicitly saved. Sessions are stateless — no server-side session tracking needed.
